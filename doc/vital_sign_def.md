# バイタルデータ　
更新日:2015/12/21
## 趣旨
本事業では拡張ストレージにバイタルデータを出力することを求めている。
ここではバイタルデータを、看護師がベッドサイドで測定し、温度板等に記録したものを指す。
以下に仕様を示す。

##バイタル検査結果通知 物理構造

    |-- 拡張ストレージ ルートフォルダ
        |-- 患者ID 先頭3文字
            |-- 患者ID 4～6文字
                |-- 患者ID
                    |-- 診療日...(*1)
                        |-- データ種別...(*2)
                            |-- コンテンツフォルダ...(*3)
                                |--主文書ファイル...(*4)

(\*1)診療日: OBX-14 検査日時の日付部分(YYYYMMDD)とする

(\*2)データ種別: L-OBSERVATIONS^OBSERVATIONS^99ZL01

(\*3)コンテンツフォルダ: 患者ID\_診療日\_データ種別コード\_特定キー\_発生日時\_診療科コード\_コンディションフラグ

(\*4)主文書ファイル: main_document.hl7
