# ICD-10不明病名
更新日: 2017/02/09

## 趣旨
本事業に参加するベンダーの電子カルテでは病名情報としてICD10を登録しているが、過去の医事会計システムからデータが移行されていない等の理由で、ICD-10が不明な場合がある。

### ICD-10が不明な場合の出力

PRB-10(プロブレムの分類)には、<ICD-10の値>^^I10^<診断種別>^<テキスト>^JHSD0004という出力が求められている。  
ICD10が不明で、PRB-10がNULLとなる場合、PRB-3(プロブレムID)は20084317^【病名】^MDCDX2と出力すること。
```
例）20084317^１型糖尿病^MDCDX2
```
20084317は未コード化傷病名（レセプト電算処理コード：0000999）に対応する病名管理である。
